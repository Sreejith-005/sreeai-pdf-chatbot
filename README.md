# 📄 SreeAI PDF Chatbot

SreeAI PDF Chatbot is a conversational AI application that allows users to upload a PDF and ask questions directly from the document using Retrieval-Augmented Generation (RAG).

The chatbot uses semantic search and conversational memory to provide accurate answers from the uploaded PDF in real time.

---

## 🚀 Features

- 📄 Upload and chat with PDF documents
- 💬 Conversational memory support
- 🔍 Semantic search using vector embeddings
- ⚡ Real-time streaming AI responses
- 🧠 Retrieval-Augmented Generation (RAG)
- 🔒 Secure API key management with Streamlit Secrets
- 🎨 Interactive Streamlit UI
- 🤖 Powered by Groq LLM and LangChain

---

## 🛠️ Technologies Used

- Python
- Streamlit
- LangChain
- FAISS Vector Database
- HuggingFace Embeddings
- Groq LLM
- Sentence Transformers

---

## 🧠 How It Works

1. Upload a PDF document
2. PDF text is split into smaller chunks
3. Embeddings are generated using HuggingFace models
4. FAISS stores vector embeddings for semantic retrieval
5. Relevant chunks are retrieved based on user questions
6. Groq LLM generates contextual answers from the retrieved content
